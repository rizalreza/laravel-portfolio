

## How to use

**Clone repo**

	git clone https://github.com/rizalreza/laravel-httpclient.git


**Database Configuration**

	* Create new database
	* Edit credentials database on .env file
		DB_DATABASENAME=
		DB_USERNAME=
		DB_PASSWORD=
Migrate

	php artisan Migrate
